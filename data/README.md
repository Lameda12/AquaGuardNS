# Data

Static fallback SST data for Nova Scotia aquaculture sites.

## Files

- `mahone_bay_sst_fallback.csv` — Mahone Bay SST 2023-01-01 to 2026-05-12.
  Format: `date,sst_celsius` (ISO date, float).
  Generated by `generate_fallback.py`. Used when NOAA ERDDAP is unreachable.

## Regenerate

```bash
cd /path/to/aquaguard-ns
uv run python data/generate_fallback.py
```
